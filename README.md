# 🎥 ZAPL Recorder (Free Edition)

Ультра-легкая программа для записи экрана и стриминга, оптимизированная специально для слабых ПК (AMD A-серии, Radeon R3 и аналоги).

### 🚀 Особенности:
* **DXGI Capture**: Максимальный FPS за счет использования библиотеки `dxcam`.
* **Микрофон**: Запись звука без рассинхрона через FFmpeg.
* **Свобода**: Никаких платных функций, лимитов по времени или принудительных водяных знаков.
* **Стриминг**: Поддержка RTMP (YouTube, Twitch).

### ⚙️ Инструкция и рекомендации:
1. **Для игр**: Обязательно ставьте режим **"Окно без рамки" (Borderless)**, иначе может быть черный экран.
2. **Настройка**: Для слабых процессоров рекомендуем выставлять **30 FPS**.
3. **Запуск**: Программа требует прав администратора для корректного захвата экрана.
